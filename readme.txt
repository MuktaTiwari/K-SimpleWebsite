# Keshav Khandelwal - Portfolio

Welcome to the personal portfolio website of **Keshav Khandelwal**. This website showcases my skills, projects, and contact information.

## Technologies Used
- **HTML**
- **CSS**


## Installation 

Follow the steps below to view the portfolio locally:

1. **Clone the repository:**

   git clone <repository-url>

2. **Navigate to the project directory:**

   cd Frontend


3. **Open the `index.html` file:**
   - You can either open it directly in your web browser or serve it using a local server for a better development experience.

---

## Project Structure

- **`index.html`**: The main HTML file that contains the structure and content of the website.
- **`style.css`**: The CSS file that provides styling for the website.
